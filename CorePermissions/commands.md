Commands
=======

Command format is /perm [noun] [target] [verb] [args...]

Any combination of what is below will allow you to structure a working permissions command.

Nouns
-----
- player
- group

Verbs
-----
- set
- unset
- purge
- prefix
- suffix
- tabcolor
- chatcolor
- view

Groups:

- create
- delete
- setparent
- removeparent
- setpriority

Players:

- addgroup
- delgroup
- setgroup
